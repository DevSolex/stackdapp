# Commit guide (40 commits, 20–30 lines each)

This repo uses **@stacks/connect** and **@stacks/transactions** throughout.

- `src/` — build contract call payloads (transactions) and connect options.
- `examples/` — deposit, withdraw, read-balance examples.
- `docs/` — API, errors, network, basis-points, uSTX.

Run `bash scripts/add-40-commits.sh` to add remaining commits, then `git push origin master`.
